Exploring the Latest Web Design Trends

On a website, micro-interactions are small animations that offer subtle feedback to users. One of the most commonly used micro-interactions is seeing a link change colors when a user mouses over it. With the focus on micro-interactions, that same experience might be given more attention to stand out.
